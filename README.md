# 📝 Task CLI

A sleek, lightweight Command Line Interface (CLI) to manage your daily tasks efficiently. Track your productivity directly from your terminal with simple commands.

---

## 🚀 Features

- **Quick Add:** Create tasks instantly with custom descriptions.
- **Dynamic Updates:** Modify task descriptions on the fly.
- **Status Tracking:** Seamlessly transition tasks between `todo`, `in-progress`, and `done`.
- **Flexible Listing:** View your entire dashboard or filter tasks by their current status.
- **Easy Removal:** Purge tasks from your list with a single command.

---

## 🛠️ Usage Syntax

```bash
task-cli <command> [arguments]
```
